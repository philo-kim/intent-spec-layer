# PRD: Coupon Orders

Users can create an order, add items, and apply a coupon before checkout.

The coupon gives a fixed discount and should be checked before payment. The
checkout page should show the amount due and let the user confirm the order.

When the payment succeeds, the order is confirmed. If the user cancels the order,
the order should be marked as cancelled.

The system should be easy to use and should show friendly messages for invalid
coupon codes.
